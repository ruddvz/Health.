  1. Step 1 — Open claude.ai in your browser (or the Claude app).
  2. Step 2 — Paste the prompt into a new conversation.
  3. Step 3 — Hit send and wait — Claude will think for about 30 seconds.
  4. Step 4 — Claude should reply with plain JSON only (no Artifact): the message starts with { and ends with }.
  5. Step 5 — Copy the entire JSON (from { through }). If Claude wrapped it in a code fence, copy only what’s inside.
  6. Step 6 — Paste into a plain text editor and save as myplan.json.
  7. Step 7 — Come back here and tap Upload My Plan above.
